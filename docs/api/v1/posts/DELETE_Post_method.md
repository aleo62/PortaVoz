# DELETE Methods from API

## DELETE Post

### URL

```
    https://portavoz.onrender.com/api/v1/posts/:postId
```

‎

### BODY

No Body

‎

### HEADERS

```
    Authorization: Bearer (current user token)
```

‎

### RETURN

```
    message: "Post deleted"
    data: <PostType>{}
```
